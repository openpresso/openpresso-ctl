**Openpresso-ctl** is the cli tool to control [openresso daemon](https://github.com/openpresso/openpressod) of the [OpenPresso Project](https://openpresso.org). 

It allows to control power state, start and stop brew process and switch to steam mode in the same way as it happens when 
a physical button on the machine is pressed. It also allows to change brew preset and user settings and provides 
metrics and events monitoring capabilities.
