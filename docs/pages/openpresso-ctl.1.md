# NAME

openpresso-ctl - Command line utility for Openpresso

# SYNOPSIS

**openpresso-ctl** [*OPTIONS*] [*COMMAND*]

# DESCRIPTION

Command line utility to control and interact with the Openpresso daemon.

# OPTIONS

See `openpresso-ctl --help` for available options.
